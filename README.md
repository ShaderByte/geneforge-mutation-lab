# GeneForge Mutation Lab

**GeneForge Mutation Lab** is a Windows desktop planner for ARK creature stat and mutation builds. It helps players compare level 1 adult/tamed stat values, assign wild stat points, preview 100% imprint results, and share saved builds.

Built by **ShaderByte**.

## Download

Use GeneForge in a mobile browser:

https://shaderbyte.github.io/geneforge-mutation-lab/

Download the Windows app directly from this repository:

- [GeneForge Mutation Lab Setup 1.1.2.exe](downloads/GeneForge%20Mutation%20Lab%20Setup%201.1.2.exe)
- [GeneForge Mutation Lab 1.1.2.exe](downloads/GeneForge%20Mutation%20Lab%201.1.2.exe)

## Features

- ASA and ASE stat mode dropdown
- Creature search with map and diet filters
- Adult/tamed base stat display
- Wild point assignment controls
- 100% imprint preview
- Remaining point counter
- Import, export, and shareable build links
- Local saved builds
- Community tab placeholder for future online uploads, ratings, and comments
- AI-style build optimizer for prompts like `lvl 362 Rex with 80K HP and the rest in melee`
- Windows desktop build with custom app icon

## Accuracy Notes

The ASA roster is filtered to normal ARK: Survival Ascended PC/console creatures. ASE mode includes creatures with matching ARK Smart Breeding ASE stat records. Bosses, alpha/brute variants, mobile rows, humans, and vehicles are intentionally excluded.

Some unusual creatures have special tamed stat adjustments. GeneForge includes handling for tamed/wild base differences and direct melee penalties such as Carcharodontosaurus and Giganotosaurus.
